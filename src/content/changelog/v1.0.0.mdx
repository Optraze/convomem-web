## Features

- **Conversation capture** — Process chat, voice, email, and SMS conversations
- **Memory extraction** — Automatic fact extraction with confidence scoring
- **PII redaction** — Sensitive data detected and masked before storage
- **Durability scoring** — Each fact scored for long-term relevance
- **Insights** — Sentiment analysis and intent detection
- **REST API** — Full API with SDK support for Node.js
- **On-prem deployment** — Available on the Custom tier

## Supported channels

- Chat (web widget, in-app)
- Voice (call transcripts)
- Email
- SMS
