ConvoMem provides official SDKs for TypeScript, Python, and Rust, plus full REST API
support via cURL.

## Install the SDK

<CodeTabs>
  <CodeTab language="typescript" filename="Terminal">
  ```bash
  # npm
  npx jsr add @optraze/convomem-sdk

  # pnpm
  pnpm i jsr:@optraze/convomem-sdk

  # bun
  bunx jsr add @optraze/convomem-sdk

  # deno
  deno add jsr:@optraze/convomem-sdk
  ```
  </CodeTab>
  <CodeTab language="python" filename="Terminal">
  ```bash
  pip install convomem
  # or with uv
  uv add convomem
  ```
  </CodeTab>
  <CodeTab language="rust" filename="Terminal">
  ```bash
  cargo add convomem
  ```
  </CodeTab>
  <CodeTab language="curl" filename="Terminal">
  ```bash
  # No installation needed — use curl or any HTTP client
  export CONVOMEM_API_KEY=sk-org-your-key-here
  ```
  </CodeTab>
</CodeTabs>

## Get an API key

Create a key from the [dashboard](https://app.convomem.com/settings/keys). Keys are
scoped per environment — use separate keys for development and production.

<Callout type="warning" title="Keep keys server-side">
Never expose an API key in client-side code. All ConvoMem calls should originate from
your backend or an edge function. For browser widgets, use [embed tokens](/docs/embed).
</Callout>

## Initialize the client

<CodeTabs>
  <CodeTab language="typescript" filename="convomem.ts">
  ```typescript
  import { ConvoMemClient } from '@optraze/convomem-sdk'

  const client = new ConvoMemClient({
    apiKey: process.env.CONVOMEM_API_KEY!,
  })
  ```
  </CodeTab>
  <CodeTab language="python" filename="convomem.py">
  ```python
  import os
  from convomem import ConvoMemClient

  client = ConvoMemClient(api_key=os.environ["CONVOMEM_API_KEY"])
  ```
  </CodeTab>
  <CodeTab language="rust" filename="main.rs">
  ```rust
  use convomem::ConvoMemClient;

  fn main() -> Result<(), Box<dyn std::error::Error>> {
      let client = ConvoMemClient::new(
          std::env::var("CONVOMEM_API_KEY")?
      );
      Ok(())
  }
  ```
  </CodeTab>
  <CodeTab language="curl" filename="Terminal">
  ```bash
  export CONVOMEM_API_KEY=sk-org-your-key-here
  ```
  </CodeTab>
</CodeTabs>

## Verify the connection

<CodeTabs>
  <CodeTab language="typescript" filename="verify.ts">
  ```typescript
  const customers = await client.customers.list({ limit: 1 })
  console.log('Connected:', customers.total >= 0)
  ```
  </CodeTab>
  <CodeTab language="python" filename="verify.py">
  ```python
  result = client.customers.list(limit=1)
  print(f"Connected: {result.total >= 0}")
  ```
  </CodeTab>
  <CodeTab language="rust" filename="main.rs">
  ```rust
  let result = client.list_customers(Some(1), Some(1)).await?;
  println!("Connected: {}", result.total >= 0);
  ```
  </CodeTab>
  <CodeTab language="curl" filename="Terminal">
  ```bash
  curl -s -H "X-API-Key: $CONVOMEM_API_KEY" \
    "https://api.convomem.com/api/v1/customers?limit=1" | jq '.total'
  ```
  </CodeTab>
</CodeTabs>

Once this succeeds, continue to the [Quick start](/docs/getting-started) to capture
your first conversation.
